The web client

The client is created using the latest version of Angular.

The application uses Angular Material combined with Bootstrap 4 for it's styling. Some custom modifications has been made also. For icons I've used Font Awesome.

The charts in the app is created with Charts.js.

The client is hosted in an AWS S3 bucket.

Server side

The current backend is written with NodeJS with TypeScript, and is created as AWS Lambdas. This is the third iteration of the back-end. Originally it was in PHP, then NodeJS with ExpressJS, and now it's moved to AWS Lambda.

For the database that stores all the item price data, recipes, items and pets it uses MariaDB.

Instead of querying the database for static data, this is stored on AWS s3 as compressed JSON files. This reduces the hosting costs, as the app don't have to waste compute resources when it does not need to. I do for now, store all auctions for the past 30 days on S3, and keep price history over the past 14 days in the database.
