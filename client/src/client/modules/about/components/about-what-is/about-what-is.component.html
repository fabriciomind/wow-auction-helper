This is a web application, developed with the intention of helping the user earn gold. Be it by crafting, snatching up bargans, checking out price trends, resetting a market, Or maybe just checking out the trade vendors.

Dashboard

The dashboard contains several panels. Each of these panels is supposed, to give some useful information.

You can define rules in the watchlist page, for that dashboard. By default, there comes a couple groups and items with rules defined. You might want to tweak these for your realm. Only the items that meet your criteria are visible on the dashboard page.

More panels may be added in the future.

Crafting

In the crafting section, you can filter recipes by profit, regional sale rate, profession etc. You can also hide any recipes you don't have; given that you have added characters to the app.

You can add more characters in character settings.

Item details and statistical item data

By clicking on an items, recipe or pet name in any table, you will get a popup with more information about that item.

Here you can get information like

  • Market value and sale rate via the TSM api
  • Your personal sale rate(if TSM addon data is imported), a chart over your past sales and purchases of that item.
  • Price and quantity statistics/history for the past 14 days
  • Sources like created by, sold by, dropped by, milled by etc
 Import of TSM addon data

Doing this will allow you to get inventory, sale, purchase and expenses stats. This information can be used to calculate crafting costs for the shopping cart etc.
